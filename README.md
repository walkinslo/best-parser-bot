## NSFW!
Telegram-бот, который отправляет фото с Rule34 по заданному тегу.
Создатель - [Walkinslo](https://github.com/walkinslo).

Команды бота:

- `/start` — приветственное сообщение.
- `/tag` — получить фото по заданному тегу.
- `/help` - помощь по командам.
